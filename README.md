# Animore 🎦
>`Animore`은 애니메이션을 즐겨 보는 사람들을 위해 애니와 관련된 영화 정보들을 한 눈에 볼 수 있도록 한 반응형 웹사이트 입니다.    
## Period ⌛
- **2024.05.25 ~ 2024.06.**
 
 

## Main Function 📌
>### **1. 애니메이션 정보 제공**  
해당 애니메이션을 검색하여 제목, 영화 상영일, 줄거리, 특전 현황과 같은 정보를 얻으실 수 있습니다.   
>### **2. 굿즈 판매 페이지**
현재 판매하고 있는 굿즈를 구경하실 수 있습니다.  <br>
>### **3. OST**
OST 표지를 클릭 시 링크를 타고 들어가 음악을 감상하실 수 있습니다.  <br>
